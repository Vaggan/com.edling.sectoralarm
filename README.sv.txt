Sector Alarm har ändrat sitt API och det gör att denna app inte längre fungerar.
Jag har varit i kontakt med dem för att se om det går att hitta en lösning men det se inte ljust ut.

Med denna app kan du larma på och av (även skalskydd) med hjälp av flows.
Det går även att trigga andra flows när larmet slås på eller av.

Instruktioner:
1. Inställningar
  a. Fyll i inloggningsuppgifter till Mina sidor och larmkod.
  b. SiteId behöver inte fyllas i, det fältet sätts automatiskt vid första inloggning.
2. Lägg sedan till ditt larm som en enhet.
